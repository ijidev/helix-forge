## Why Helix-Forge?

- **Explicit over Magic** — No facades, no `__call`, no global helpers. Everything traceable.
- **Attribute-Driven** — Routing, validation, DI, permissions via PHP 8.3 `#[Attributes]`.
- **Schema-First** — The code is the source of truth. No scattered config files.
- **Zero Bloat Core** — Core is <2MB. Everything else is a Composer package.
- **AI-Native** — Agent protocol so AI can safely generate, modify, and test code.
